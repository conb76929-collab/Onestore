🌌 OneStore | Quantum OS Drive

OneStore is a high-performance, private-first cloud storage simulation running entirely in the browser. It combines a sleek, "Nebula-grade" interface with persistent local storage and a massive suite of engineering controls.
🚀 Core Features

    Permanent Vault: Utilizing IndexedDB technology, files are "burned" into your browser's local ledger for persistent access—no login required.

    50 Quantum Plans: Instantly switch between 50 distinct storage tiers (from 1TB to 500PB), all unlocked and free for local use.

    Engineering Deck: Over 40+ configurable settings across UI, Security, Networking, and Kernel categories.

    Secure Ingest: A drag-and-drop terminal for file ingestion with high-speed binary reconstruction.

    Action Terminal: Click any file in your archive to instantly Download or Delete with one-tap verification.

🛠️ Technical Architecture

OneStore is built as a Client-Side Heavy (CSH) application.

    Frontend: Pure HTML5, CSS3 (with Glassmorphism and Nebula themes), and Vanilla JavaScript.

    Icons: Powered by Lucide.

    Database: IndexedDB (OneStore Reinforced V5 Kernel).

    Data Handling: Asynchronous Base64 binary processing to prevent "Undefined" path errors.

    [!IMPORTANT]
    Privacy Note: This drive operates on a "Zero-Server" model. All data is stored locally on your machine. Your files never touch a central server, making this 100% private and invisible to the public internet.
